Chybové kódy

404 Not Found

Chyba 404 znamená, že server nemôže nájsť požadovanú stránku. Toto je najčastejšia chyba, ktorú používatelia vidia.

403 Forbidden

Chyba 403 znamená, že prístup k požadovanej stránke je zakázaný. Server pochopil požiadavku, ale odmieta ju splniť.

500 Internal Server Error

Chyba 500 je všeobecná chybová správa, keď server narazí na nečakanú situáciu a nevie, ako pokračovať.

502 Bad Gateway

Chyba 502 znamená, že server, cez ktorý prebieha požiadavka, dostal neplatnú odpoveď.

503 Service Unavailable

Chyba 503 znamená, že server momentálne nie je schopný spracovať požiadavku. Môže to byť spôsobené preťažením servera alebo údržbou.

504 Gateway Timeout

Chyba 504 znamená, že server, cez ktorý prebieha požiadavka, neodpovedal včas.
