Web Dizajn

Web dizajn zahŕňa mnoho rôznych zručností a disciplín pri tvorbe a údržbe webových stránok. Rôzne oblasti web dizajnu zahŕňajú grafický dizajn; dizajn rozhrania; autorské práva, vrátane štandardizovaného kódu a vlastného softvéru; dizajn používateľskej skúsenosti; a optimalizácia pre vyhľadávače.

HTML

HTML (HyperText Markup Language) je najzákladnejším stavebným blokom webu. Definuje význam a štruktúru webového obsahu. Okrem HTML sa na popis vzhľadu / prezentácie webovej stránky (CSS) alebo funkčnosti / správania (JavaScript) zvyčajne používajú iné technológie.

CSS

Cascading Style Sheets (CSS) je jazyk štýlových listov používaný na popis vzhľadu a formátovania dokumentu napísaného v HTML. CSS je primárne navrhnuté na umožnenie oddelenia obsahu dokumentu od prezentácie dokumentu, vrátane prvkov ako je rozloženie, farby a písma.
