Chybové kódy

Chybové kódy HTML sú štandardizované číselné identifikátory, ktoré webové servery používajú na komunikáciu o typoch problémov, ktoré môžu nastať pri spracovaní požiadavky na HTTP. Napríklad, "404" je chybový kód, ktorý znamená "požadovaná stránka nebola nájdená".

Web Dizajn

Web dizajn je proces tvorby webových stránok. Zahrňuje aspekty ako layout, grafický dizajn, obsahovú produkciu a dizajn grafického rozhrania. Web dizajn je technická oblasť, kde dizajnéri používajú HTML, CSS, JavaScript a ďalšie nástroje na vytvorenie vizuálne a interaktívne webové stránky.
